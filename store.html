Pennsylvania Map of 2020 Election Results

The problem for this project was to create a map of Pennsylvaniaâ€™s 2020 presidential election votes with gradation of color depicting the percentage of votes for either Biden or Trump. In order to create this choropleth map, I joined two shapefiles: one with county data and one with voting data. I then exported the counties that Biden won to its own file and the counties that Trump one to another file. I adjusted the values to the necessary percentages by which each candidate won to the desired color (lightest for the tightest races and darkest for the easiest wins of blue and red respectively). I then added the extra features like the scale bar and title to the map. GIS Tools Used: Add Joins, Select by Attributes, Labeling, Choropleth Maps
