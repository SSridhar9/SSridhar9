A Map of Homicides in Chicago

The problem for this lab assignment was to describe the distribution of homicides across police districts in Chicago. I first separated the homicides from other crimes in the file by selecting for homicides. After that, I added the “area” field to the police district file so I knew the size of each of the districts. I then joined the police district and crimes file so the density of homcides can be found using the district size information I just added. I had ArcGIS calculate the number of homicides per square mile in each police district, which I then used to make my choropleth map which is basically a density map. GIS Tools Used: Adding Joins, Creating Summary Tables, Calculating Fields, Choropleth Maps
